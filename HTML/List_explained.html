HTML Lists

This page shows three types of HTML lists: Unordered List, Ordered List, and Description List.

Unordered List (ul)

  • HTML
  • CSS
  • JavaScript

Use an unordered list when the order of items is not important.

Ordered List (ol)

  1. Open the browser
  2. Type the website address
  3. Press Enter

Use an ordered list when steps or sequence matter.

Description List (dl)

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
Browser
A software used to view web pages

Use a description list when each item needs an explanation.
