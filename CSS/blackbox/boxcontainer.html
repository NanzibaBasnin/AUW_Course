The CSS Box Model

Every HTML element is a rectangular box with 4 layers

Content

Text, images, or videos

Width: 300px

Height: 200px

Content + Padding + Border

See all three layers visually!

With Margin

Notice the space outside the border!

1. Content

The actual stuff inside: text, images, etc.

Set with width and height

2. Padding

Space INSIDE, between content and border

Same background color as the element

3. Border

The visible edge around padding

Has style, width, and color

4. Margin

Space OUTSIDE, between elements

Always transparent
