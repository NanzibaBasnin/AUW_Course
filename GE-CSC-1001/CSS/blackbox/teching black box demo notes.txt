Box #1: Solid black rectangle (content only)

Box #2: Black box with gray center (content + padding)

Box #3: Black box, red border, gray, white center (all 3 layers)

Box #4: Same as #3 but with more space around (adds margin)

/* DEMONSTRATE THIS LIVE in class: */

/* 1. Start with just content */
.box {
    width: 300px;
    height: 150px;
    background: black;
}

/* 2. Add padding (box grows INWARD) */
.box {
    padding: 30px;  /* Content gets smaller visually */
}

/* 3. Add border (box grows OUTWARD) */
.box {
    border: 10px solid red;  /* Adds to total size */
}

/* 4. Add margin (space around box) */
.box {
    margin: 20px;  /* Doesn't change box size, just position */
}

Key Points to Emphasize:
box-sizing: border-box is your friend

Padding = inside space (same color as element)

Border = visible edge (any color/style)

Margin = outside space (always transparent)

Order from inside out: Content → Padding → Border → Margin
